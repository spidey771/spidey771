## 📅 Research Timeline

<br/>

**🟦 2014 — Network & Server Basics** `Age 9`
> First exposure to networking through Minecraft server hosting.
> Learned **port forwarding**, **TCP/UDP protocols**, and basic server architecture independently.

---

**🟦 2015 — Server Development & Plugin Engineering** `Age 10`
> Operated a mid-scale Minecraft server.
> Developed **Java-based server plugins**, built **PHP web integrations**, and implemented log collection systems.

---

**🟦 2015–2016 — Reverse Engineering Introduction** `Age 10–11`
> Began analyzing game memory structures using **Cheat Engine**.
> Learned **hardware breakpoints**, **CPU register architecture**, and **DBVM (virtualization-based debugging)**.
> Researched signature-based protection bypass techniques.

- Tools: `Cheat Engine` `OllyDbg`

---

**🟦 2018 — Kernel Driver Development** `Age 13`
> Developed first **Windows kernel-mode driver (Ring0)** from scratch.
> Implemented **kernel↔usermode communication via IOCTL**.
> Analyzed encrypted pointer structures (UWorld, GName) in a game client,
> wrote **MASM shellcode** to hook decryption routines and retrieve plaintext pointers.
> Built overlay rendering (ESP) via GDI hooking.

- Stack: `C` `C++` `MASM` `WDM Driver` `IDA Pro`

---

**🟦 2019–2020 — Game Engine Architecture Analysis** `Age 14–15`
> Reverse engineered a game's **ECS (Entity Component System)** architecture.
> Traced `EntityAdmin → Component Base (Parent) → Component ID` access chain.
> Decrypted parent pointers and enumerated player entities to extract:
> skeleton data, HP, rotation, velocity.
>
> Also implemented a **pixel-pattern-based aimbot** using screen scanning and
> kernel-level mouse input control via DD driver.

- Stack: `C++` `IDA Pro` `x64dbg`

---

**🟦 2021 — Vulnerability Research & Packet Analysis** `Age 16`
> Independently discovered an **IP address disclosure vulnerability** in a major online game client.
> Captured and analyzed packets, reverse engineered ~80% of the exploit mechanism.
> **Analysis writeup was covered by a domestic IT news outlet.**

- Tools: `Wireshark` `x64dbg` `Python`

---

**🟦 2021–2022 — Injection Technique Research** `Age 16–17`
> Researched injection bypass techniques against a highly hardened anti-cheat system:
> - Located **signed RWX memory regions** for shellcode staging
> - Analyzed `Present()` hook point in an overlay DLL
> - Implemented **Remote Shellcode Injection** to bypass signature verification
>
> Additionally analyzed an **EasyAntiCheat loader vulnerability**:
> - Reversed EAC initialization routine in game launcher
> - Discovered conditional branch (`JNE`) modification skips driver loading
> - Analyzed heartbeat packet structure for protocol-level bypass possibility

- Stack: `C++` `x64 ASM` `PE Analysis` `Wireshark`
